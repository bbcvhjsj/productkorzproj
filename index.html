Проект "Бумеранг добра"

Проект "Бумеранг добра" это боксы по приёму вещей для нуждающихся семей. Как проходит рабочий процесс боксов? - спросите вы. Бумеранг добра разгружается по мере заполнения, в дальнейшем сортируются у нас на складе. В котором мы производим раздачу вещей для всех нуждающихся семей. Где находятся данные боксы? Изменения в локациях наших боксов Бумеранг Добра Наш 1 бокс находится по адресу Рыскулова 190 в здании типографии ныне Актобе Медиа, центральный вход. Наш 2 бокс находится по адресу 11 мкр 112Г, ТЦ Актобе Ажары, 3ий этаж. В ТЦ Актобе мол больше нет нашего бокса.

Проект «Бумеранг Добра»

Добрый день, уважаемые актюбинцы! Мы начинаем с супер бомбических новостей. Частный благотворительный фонд «Фонд помощи продуктовая корзина» @fond_pomoshi_aqtobe установил 2 бокса «Бумеранг Добра». У нас появились новые локации боксов. Боксы — бумеранг добра, которые рассчитаны на прием продуктовой помощи для нуждающихся семей. Одна из точек по новой локации расположена по адресу: Ул 101 дом 16 магазин Azia Shop @azia.icecream.kz работает до 19-00 без выходных. КР "Табыс", проспект Санкибай батыра, 24н. Центральный главный вход. @rynok_tabys_bazary Помощь которая в дальнейшем будет доставлена и выделена для нуждающихся семей. Любой желающий может оставить не скоропортящиеся продукты, став вкладом в отворительность. Мы рассчитываем на вашу поддержку друзья, вместе мы сила.
